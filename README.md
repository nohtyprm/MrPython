# MrPython

A simplified programming environment for Python (3.x) - based on IDLE

## Installation

Git command line:

```
$ git clone --recurse-submodules https://github.com/fredokun/MrPython.git
...
$ cd MrPython
$ python3 mrpython/Application.py &
```
